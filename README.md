
# Rock-Paper-Scissor Game

A simple rock-paper-scissor game web app using VueJS.


## Run Locally

Clone the project

```bash
  git clone https://github.com/vpandey6398/rock-paper-scissor.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
